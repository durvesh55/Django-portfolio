# 🚀 Django Portfolio Website

A full-stack portfolio website built using Django with an integrated Password Manager.

## 🔥 Features

* Responsive modern UI (Bootstrap + custom CSS)
* Contact form with backend storage
* Django Admin panel integration
* Password Manager (CRUD operations)
* Password hide/show toggle 👁
* Copy password feature 📋

## 🛠 Tech Stack

* Python (Django)
* HTML, CSS, Bootstrap
* SQLite
* JavaScript
